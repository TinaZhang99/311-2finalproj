Description of Files in Repository
By: Tina Zhang

1. 
"RDATA els_02_12_byf3pststu_v1_0.zip" was downloaded from the National Center 
for Educational Statistics (NCES) website and contains the complete set of data
used in this project, in R format

2. 
"Codebook.txt" is the codebook for the variables analyzed in this project,
also downloaded from the NCES website.

3.
"final project analysis final.Rmd" contains the R code used for analysis in
this project, including cleaning the data and performing lasso

Additional data and information associated with the Education Longitudinal
Study of 2002 (ELS:2002) can be obtained here: 
https://nces.ed.gov/pubsearch/pubsinfo.asp?pubid=2015314
